
* Quake 3 Map Renderer

Loads BSP files and renders them.

Right now only a few BSP features are handled.


* How to Run?
#+BEGIN_SRC: bash
lein run
#+END_SRC

* Controls
+ Click and drag mouse to position viewport.
+ Use keys w & s to move around.
